Instead of calling directly $app['translator']->trans(), we check for the presence of a placeholder named '%contenttype%'.

If one is found, we replace it with the contenttype.name parameter, and try to get a translated string. If there is not, we revert to the generic (%contenttype%) string, which must have a translation.

Special parameter keys: 'DEFAULT': the value is returns instead of the key of no translation is found 'NUMBER': transCjoice is triggered with the value as countvalue
